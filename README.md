## Syntax

### Variables
```
int int_variable = 2
string str_variable = "string variable"
```
Now, variable's types need to be declared <br>
This will be changed soon

### Functions

```
func function_name()
{
    //code
}
```
Parameters are not available now <br>
They will be added soon too.

### Internal functions

```
print("hello world!")
```
